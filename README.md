# AppsScript_Gmail_Anexos
Script do Google Apps para salvar anexos do Gmail com assunto específico no Drive
Visão Geral: O script executa uma query em sua conta Gmail buscando por assunto específico definido por você no script, realiza o download do arquivo nesse e-mail e salva em uma pasta no seu Google Drive através do ID_folder
Pré-requisitos: Ter uma conta Google/Gmail.

Instalação/Configuração (Crucial):

Etapa 1: Criar Pastas no Google Drive e obter seus IDs.
Etapa 2: Copiar o código Code.gs para um novo projeto no script.google.com.
Etapa 3: Configurar os objetos CONFIGURACOES e EMAIL_NOTIFICACAO.
Etapa 4: Autorizar o script.
Etapa 5: Configurar o Gatilho Semanal (Exemplo:Toda segunda-feira, 12h-13h).
